PROJECTS

Machine learning
Applied research project as a part of the MA thesis: Gathered the data, using Python and Jupyter notebooks built a dataset, and then built a CNN machine learning model for Mandarin Chinese tones pronunciation evaluation for second language learners. There are three models, all are using features based on Mel-Frequency Cepstral Coefficients. One supplementary model is used to detect speaker’s gender, while two others are trained to tone pronunciation of the two different genders. There are also generated visualizations of the voice fundamental frequency of the input recording for further research. Deployed online as a web-app with streamlit for front/back end.
